## Data Source

The dataset used in this project is from Kaggle:

Stock Prices Dataset (5 Years):  
https://www.kaggle.com/datasets/sherrytp/stock-prices-5y

Please download the dataset and place the file `all_stocks_5yr.csv` in the project root directory before running the model.
 

##  How to Run：

1\. Run the model

python XGBoost.py

2\. Run the dashboard

streamlit run stock\_dashboard/app.py



## Requirements：

Install the required packages:

pip install pandas numpy matplotlib scikit-learn xgboost streamlit



## Output Files：

After running the model, prediction results and summary metrics will be saved in:

stock\_dashboard/results/
